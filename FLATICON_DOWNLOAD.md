# Download Icon dari Flaticon

## Langkah Download:

### 1. Instagram Icon
- Link: https://www.flaticon.com/free-icons/instagram-logo
- Klik "Download PNG" (pilih ukuran 64px atau 128px)
- Simpan sebagai: `instagram.png` di folder `public/icons/`

### 2. Menu/Fast Food Icon
- Link: https://www.flaticon.com/free-icons/fast-food
- Klik "Download PNG" (pilih ukuran 64px atau 128px)
- Simpan sebagai: `menu.png` di folder `public/icons/`

### 3. Location/Map Icon
- Link: https://www.flaticon.com/free-icons/map-location
- Klik "Download PNG" (pilih ukuran 64px atau 128px)
- Simpan sebagai: `location.png` di folder `public/icons/`

### 4. WhatsApp Icon
- Link: https://www.flaticon.com/free-icons/whatsapp
- Klik "Download PNG" (pilih ukuran 64px atau 128px)
- Simpan sebagai: `whatsapp.png` di folder `public/icons/`

## Struktur File Akhir:
```
public/
  └── icons/
      ├── menu.png          (Fast food icon)
      ├── location.png      (Map location icon)
      ├── instagram.png     (Instagram logo icon)
      └── whatsapp.png      (WhatsApp icon)
```
