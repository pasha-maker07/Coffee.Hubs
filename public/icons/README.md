# Icon Setup Instructions

## Langkah-langkah menggunakan icon dari Flaticon:

### 1. Download Icon dari Flaticon
Kunjungi https://www.flaticon.com/ dan download icon berikut:

- **Menu Icon** - Search: "menu" atau "food menu"
  - Simpan sebagai: `menu.png`
  
- **Location Icon** - Search: "location" atau "map pin"
  - Simpan sebagai: `location.png`
  
- **Instagram Icon** - Search: "instagram"
  - Simpan sebagai: `instagram.png`
  
- **Phone/WhatsApp Icon** - Search: "phone" atau "whatsapp"
  - Simpan sebagai: `phone.png`

### 2. Rekomendasi Icon Packs dari Flaticon:
- [Free Social Media Icons](https://www.flaticon.com/packs/social-media)
- [Free Coffee Shop Icons](https://www.flaticon.com/packs/coffee-shop)
- [Free Location Icons](https://www.flaticon.com/packs/location)

### 3. Simpan Icon
Letakkan semua file icon di folder: `public/icons/`

### 4. Format Icon
- **Format**: PNG dengan background transparan (recommended)
- **Ukuran**: 64x64px atau 128x128px
- **Warna**: Pilih warna yang sesuai atau putih (akan di-styling dengan CSS)

### 5. Styling
Icon akan otomatis di-styling oleh DaisyUI dengan:
- Background gradient (coklat)
- Ukuran responsif
- Hover effects

## Struktur File:
```
public/
  └── icons/
      ├── menu.png
      ├── location.png
      ├── instagram.png
      └── phone.png
```

## Tips:
- Gunakan icon dengan style yang konsisten
- Pilih icon yang simpel dan mudah dikenali
- Jika menggunakan SVG, pastikan viewBox sudah benar
